### Dynamic Prompt Assembly

The true power of CopyZap lies in its **layered prompt architecture**—a sophisticated system that merges all inputs into a coherent, strategically optimized prompt that produces predictable, high-quality results.

#### Prompt Hierarchy (5 Layers)

**Layer 1: System Prompt (AI Foundation)**
- Establishes AI's role as expert copywriter
- Defines output format (plain text vs. structured JSON)
- Sets critical requirements (word count enforcement, no meta-commentary)
- Includes section-specific guidance (e.g., Hero Section = attention-grabbing headline + value proposition)
- Embeds tone instructions based on tone level (formal → casual scale)
- Adds writing style preferences (persuasive, storytelling, etc.)
- Enforces language constraints (passive voice avoidance, simplicity, etc.)

**Layer 2: User Content & Context**
- Business description or original copy
- Target audience and pain points
- Key message and brand values
- Keywords and context
- Competitor information
- Product/service name and industry

**Layer 3: Mode Selection (Create vs. Improve)**
- **Create Mode:** "Build new compelling marketing copy from scratch"
- **Improve Mode:** "Enhance existing copy while maintaining core message"

**Layer 4: Tone, Persona, and Word Count**
- Language selection (English, Spanish, French, German, Italian, Portuguese)
- Tone (Professional, Friendly, Bold, Minimalist, Creative, Persuasive)
- Persona override (Steve Jobs, Seth Godin, Alex Hormozi, Brené Brown, etc.)
- Exact word count requirements with tolerance settings
- Elaboration instructions (add examples, case studies, statistics)

**Layer 5: Special Instructions (Highest Priority)**
- User's free-form directives
- Overrides any conflicting instructions from previous layers
- Ensures ultimate control and customization

#### How It Works: Example

User inputs:
- Business: "AI-powered project management tool for remote teams"
- Target Audience: "Tech startup founders and product managers"
- Tone: "Bold"
- Word Count: 300 words
- Special Instructions: "Emphasize security and compliance features"

**Assembled Prompt (Simplified):**

```text
SYSTEM PROMPT:
You are an expert copywriter. Create compelling new marketing copy.
The copy must be EXACTLY 300 words.
Use Bold tone with confident, no-nonsense language.
Focus on the Hero Section with attention-grabbing value proposition.
Output ONLY the marketing copy—no explanations or meta-commentary.

USER CONTENT:
Business: AI-powered project management tool for remote teams
Target Audience: Tech startup founders and product managers
Key Message: Streamline collaboration without compromising security

SPECIAL INSTRUCTIONS:
Emphasize security and compliance features

→ AI generates precise, on-brand copy meeting all requirements
```

- **Meta Descriptions** (1-5 variants)

  - Compelling search result snippets that drive clicks

- **H1 Variants** (1-5 variants)
  - Character limit: 60 characters
  - Primary page headlines optimized for search and user intent

- **H2 Headings** (1-10 variants)
  - Character limit: 70 characters
  - Subheadings for content structure and keyword targeting

- **H3 Headings** (1-10 variants)
  - Character limit: 70 characters
  - Supporting headings for deep content hierarchy

- **Open Graph Titles** (1-5 variants)
  - Character limit: 60 characters
  - Social media share titles (Facebook, LinkedIn, etc.)

- **Open Graph Descriptions** (1-5 variants)
  - Character limit: 110 characters

**Character Limit Enforcement:**
The AI is explicitly instructed to NEVER exceed character limits. If content approaches limits, it's automatically shortened while preserving meaning and keyword relevance. This ensures all metadata is platform-compliant and displays correctly.

#### GEO (Generative Engine Optimization)

GEO optimizes content specifically for AI assistants like ChatGPT, Claude, and Gemini—ensuring your content is easily discoverable, quotable, and surfaced in AI-generated responses.

**When GEO is Enabled:**

1. **TL;DR Summary** (Optional, recommended)
   - Automatically adds a concise one-sentence summary at the top
   - Format: "TL;DR: [Direct answer to main question]"
   - Makes content instantly extractable by AI assistants
   - Increases likelihood of being quoted in AI responses

2. **Location Targeting**
   - Naturally incorporates geographical context when specified
   - Examples: "Serving businesses in Mexico", "Helping companies across LATAM thrive"
   - Region-specific phrasing without keyword stuffing

3. **Global/Location-Neutral Content**
   - If no location specified, creates universally relevant content
   - Avoids geographical references while maintaining discoverability

**GEO Scoring System (0-100 points):**

After generation, CopyZap evaluates content across 7 criteria:

1. **Direct Answer Clarity** (20 points)
   - Does content start with a clear answer or summary?
   - Can AI assistants easily extract the main answer?

2. **Scannable Structure** (15 points)
   - Bullet points, short paragraphs, clear subheadings
   - Easy to scan and digest quickly

3. **Question-Based Headings** (10 points)
   - Headings phrased as questions AI can index
   - Match common user queries

4. **Local Relevance / GEO Markers** (20 points)
   - Mentions of specific locations, regions, geographical markers
   - Full points awarded if content is intentionally global/location-neutral

5. **Quote-Friendly Sentences** (15 points)
   - Short, clear sentences AI can easily quote
   - Standalone statements that make sense out of context

6. **Authority Signals** (10 points)
   - Examples, statistics, credentials
   - Social proof and expertise indicators

7. **Optional TL;DR / Answer Box** (10 points)
   - Proper inclusion of TL;DR when enabled
   - Answer-box style beginning

**Actionable Suggestions:**
If score is below 80, AI provides specific recommendations to improve GEO optimization.

### Retry Logic for Word Count Adherence

CopyZap includes **intelligent revision logic** to ensure generated content meets exact word count requirements:

#### Word Count Modes

1. **Standard Mode**
   - Target word count with ~20% tolerance
   - Flexible for natural phrasing
   - Prioritizes quality over exact count

2. **Prioritize Word Count Mode**
   - Strict adherence to exact word count
   - Automatic revision if outside tolerance
   - Uses multi-stage refinement process

3. **Little Word Count Mode (Short Content)**
   - For content ≤150 words
   - Flexible range (e.g., 50-100 words instead of exact 75)
   - Maintains natural flow while staying within bounds

#### Automatic Revision Process

**First Generation:**
- AI generates content targeting exact word count
- System checks actual word count against target

**Revision Trigger:**
- Content falls outside acceptable tolerance range
- Example: Target 300 words, tolerance 20%, range 240-360 words
- If content = 220 words → triggers revision

**Revision Strategy:**

**If Too Short:**
- Add detailed examples and elaboration
- Include case studies and statistics
- Expand explanations with supporting evidence

**If Too Long:**
- Remove unnecessary adjectives and filler
- Condense redundant explanations
- Tighten sentence structure

**Multi-Stage Refinement:**
1. First Revision: Standard adjustment
2. Second Revision: Enhanced parameters if still outside range
3. Emergency Revision: Aggressive adjustment if needed
