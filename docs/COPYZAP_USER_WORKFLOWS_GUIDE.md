## Overview

### What CopyZap Is

CopyZap is an AI-powered copywriting platform that helps you create, improve, and manage marketing content across multiple formats and platforms. It uses advanced AI models to generate professional copy tailored to your business, audience, and brand voice.

**Key Capabilities:**
- Generate marketing copy from scratch or improve existing text
- Fast text editing and reply generation (CopySnap)
- Multi-step content workflows for complex projects
- Template system for repeatable processes
- Brand voice management for consistency
- Export to multiple formats (Markdown, HTML)
- Usage tracking and session management

### Who It Is For

CopyZap is designed for:
- **Marketing agencies** managing multiple clients with distinct brand identities
- **Web agencies** creating website copy and landing pages
- **Marketers and founders** producing campaign materials, social posts, and emails
- **Content creators** who need AI assistance while maintaining control over output
- **Consultants** requiring professional copy for proposals and presentations

### What Problems It Solves

- **Blank page syndrome:** Start with guided wizards or proven templates
- **Brand consistency:** Define and reuse brand voices across all content
- **Time efficiency:** Generate multiple variations quickly, refine with one click
- **Platform optimization:** Platform-specific formatting for X/Twitter, LinkedIn, email
- **Workflow repeatability:** Save successful processes as templates for future use
- **Quality control:** Built-in scoring, comparison, and refinement tools

### What CopyZap Is NOT

- **Not a content management system:** No publishing or scheduling features
- **Not a collaboration platform:** Single-user editing (no real-time co-editing)
- **Not a social media scheduler:** Generates content but doesn't post to platforms
- **Not SEO software:** Generates SEO metadata but doesn't track rankings
- **Not mobile-optimized:** Desktop browser required for full functionality

## Start Hub: Ways to Begin

The Start Hub appears when you open Copy Maker (or can be manually opened). It offers three pathways to begin creating content:

### 1. Quick Prompt Wizard (Recommended for New Users)

**When to use:**
- You're new to CopyZap
- You want guidance through the generation process
- You need help deciding what inputs to provide

**How it works:**

**Step 1: Choose Mode**
- **Create New Copy:** Generate content from scratch
- **Improve Existing Copy:** Refine and enhance text you already have
- **Quick Polish:** Fast improvement mode with minimal configuration

**Step 2: Provide Context (Create mode)**
- Describe your business/project in a few sentences
- Provide sample text or URLs for context (optional)
- Specify target audience

**Step 2: Paste Content (Improve mode)**
- Paste the text you want to improve
- System analyzes content and suggests improvements

**Step 3: Configure Output**
- Select tone (Professional, Casual, Authoritative, etc.)
- Choose output length (Short, Medium, Long, or specific word count)
- Pick output structure (Paragraph, Bulleted List, Structured Sections)

**Step 4: Generate**
- Review your configuration summary
- Click "Generate Copy"
- Wizard closes and generation begins

**What happens next:**
- You're taken to the main Copy Maker interface
- Your wizard selections are pre-filled
- Generation starts automatically or waits for final confirmation
- Results appear in the Results Panel when complete

### 2. Load from Templates

**When to use:**
- You have a specific content type in mind (blog post, email, ad copy)
- You want proven structures and prompts
- You need consistent formatting across multiple pieces

**How it works:**

**Browse Templates:**
- View all available templates (yours + public templates)
- Categories: Blog Posts, Social Media, Ads, Emails, Website Copy, Miscellaneous
- Each template shows: name, description, category, public/private status

**Filter and Search:**
- Filter by category
- Search by keyword
- View template details before loading

**Load Template:**
- Click "Use Template"
- Template pre-fills all Copy Maker fields
- You can edit any field before generating
- Template does NOT auto-generate — you control when to click "Generate"

**What gets pre-filled:**
- Brief description
- Target audience
- Output structure
- Tone and style preferences
- Special instructions
- Optional features (SEO, scoring, etc.)
- All customization fields specific to that template

### 3. Start From Scratch

**When to use:**
- You have specific requirements not covered by templates
- You want full control over every input field
- You're experimenting or creating a custom workflow

**How it works:**
- Click "Start From Scratch"
- Start Hub closes
- You're taken to an empty Copy Maker form
- All fields are blank — you fill in exactly what you need

**Minimum required inputs:**
- Brief description (what you're creating)
- Product/service name (context for AI)
- Target audience (who you're writing for)

**All other fields are optional** but affect output quality and style.

## Copy Maker: Create & Improve Content

Copy Maker is the full-featured content generation engine. It offers 80+ configuration options for precise control over AI output.

### Create vs Improve Mode

**Create Mode (default):**
- Generate content from scratch
- Requires business context, audience, and desired output
- No existing copy needed
- Used for: new landing pages, blog posts, emails, ads, social posts

**Improve Mode:**
- Refine existing copy
- Paste your current text into "Original Copy" field
- AI analyzes and enhances based on your instructions
- Used for: editing drafts, refreshing old content, A/B test variations

**Toggle:** Click the "Create" / "Improve" toggle at the top of the form.

---

### Input Configuration: Required Fields

#### 1. Brief Description
**What it is:** A short summary of what you're creating.
**Examples:**
- "Landing page for SaaS project management tool"
- "Cold email for B2B lead generation"
- "LinkedIn post about AI in marketing"

**Purpose:** Gives AI high-level context. Acts as the "title" of your session.

#### 2. Product/Service Name
**What it is:** The name of what you're promoting or discussing.
**Examples:**
- "TaskFlow Pro"
- "Our consulting services"
- "AI-powered copywriting"

**Purpose:** Ensures AI references your product correctly throughout the copy.

#### 3. Target Audience
**What it is:** Who will read this content.
**Examples:**
- "B2B SaaS founders, age 30-50, technical background"
- "Busy moms looking for meal planning solutions"
- "Marketing managers at mid-size companies"

**Purpose:** Tailors language, tone, and messaging to resonate with specific readers.

---

### Input Configuration: Core Content Fields

#### Business / Project Context
**Description:** Detailed information about your business, product, or project.
**Character Limit:** 10,000 characters
**Purpose:** Provides deep context for AI to generate relevant, accurate copy.

**What to include:**
- What your product/service does
- Key features and benefits
- Unique selling proposition (USP)
- Problem it solves
- How it works (briefly)

**Tip:** More context = better output. Don't skimp here.

#### Key Message
**Description:** The single most important thing you want readers to understand or do.
**Examples:**
- "Sign up for our free trial today"
- "We solve [specific problem] better than anyone else"
- "This guide will save you 10 hours per week"

**Purpose:** Ensures AI focuses the copy on your primary goal.

#### Call to Action (CTA)
**Description:** The specific action you want readers to take.
**Examples:**
- "Book a demo"
- "Download the guide"
- "Join the waitlist"
- "Share your thoughts in the comments"

**Purpose:** Directs AI to include clear, compelling CTAs in the output.

#### Unique Selling Points (USPs)
**Format:** Comma-separated list
**Examples:**
- "30-day money-back guarantee, No credit card required, Setup in 5 minutes"

**Purpose:** Highlights competitive advantages AI should emphasize.

#### Key Features / Benefits
**Description:** What your product does and why it matters.
**Format:** Free text or bullet points
**Purpose:** Gives AI material to explain value proposition.

### Input Configuration: Output Control

#### Target Word Count
**Options:**
- Short (100-300 words)
- Medium (300-600 words)
- Long (600-1000 words)
- Custom (specify exact number)

**AI Decide Word Count Toggle:** If enabled, AI determines optimal length based on content requirements.

**Prioritize Word Count Toggle:** If enabled, AI strictly enforces word count (may make multiple revision attempts).

**Word Count Tolerance:** Percentage variance allowed (default: 10%).
Example: Target 500 words, 10% tolerance = accept 450-550 words.

**Purpose:** Controls output length to fit platform constraints or content strategy.

#### Output Structure
**Options:**
- **Single Paragraph:** Flowing prose, no breaks
- **Multiple Paragraphs:** Standard article structure
- **Bulleted List:** Key points as bullets
- **Numbered List:** Sequential steps or priorities
- **FAQ Format:** Question/answer pairs
- **Structured Sections:** Custom sections with labels and word counts

**Structured Sections (Advanced):**
- Click "Add Section"
- Define: Section Label, Target Word Count, Special Instructions (optional)
- AI generates each section independently
- Useful for: landing pages (Hero / Features / Testimonials / CTA), blog posts (Intro / Body / Conclusion)

**Purpose:** Formats output to match final use case.

#### Language
**Auto-detect:** AI detects language from your inputs (default)
**Manual Selection:** Choose from 50+ supported languages
**Purpose:** Generate content in non-English languages.

### Generation & Results

#### Starting Generation

**Click "Generate Copy"** button (bottom-right of form or floating bar).

**What happens:**
1. Pre-generation validation runs (checks required fields, placeholders, word count ranges)
2. If validation fails, error toast appears — fix issues and retry
3. If validation passes, loading overlay appears
4. Progress messages show AI processing steps
5. Credits are deducted from your balance
6. Results appear in the Results Panel when complete

**Generation Time:** 10-60 seconds depending on:
- Complexity of request
- Selected AI model
- Number of optional features enabled
- Output length

#### Validation Warnings

**Placeholder Detection:**
If your inputs contain unresolved placeholders like `[CLIENT_NAME]` or `{{VARIABLE}}`, a warning modal appears:
- Shows list of detected placeholders
- Options: "Proceed Anyway" or "Go Back and Fix"

**Purpose:** Prevents generating copy with template variables still in place.

**Insufficient Credits:**
If you don't have enough credits, a modal appears:
- Shows current balance and cost of generation
- Prevents generation until credits are available or grace period applies

### Validation Failure & Repair

**When it happens:** AI generates output that fails structural validation checks.

**Validation Checks:**
- Output exists and is not empty
- Required sections present (if using structured format)
- Alternatives exist (if requested)
- SEO metadata exists (if requested)
- No unresolved placeholders
- Language consistency

**Automatic Repair:**
If validation fails:
1. System automatically sends a "repair" request to AI with error details
2. ONE repair attempt is made
3. Repair attempt is charged as separate operation: `generate_copy_repair`
4. Effectively doubles credit cost if repair is needed

**If repair succeeds:** Repaired output is shown, session is saved normally.

**If repair fails:**
- Warning banner appears at top of results
- Shows validation errors (up to 3 displayed)
- Buttons: "Show Raw Output," "Copy Raw Output," "Regenerate"
- Session is NOT saved (prevents broken content in database)
- Raw AI response available for debugging

**User actions:**
- **Show Raw Output:** Opens modal with unprocessed AI response
- **Regenerate:** Clears error state and starts fresh generation
- **Modify Inputs:** Adjust inputs and try again

**Purpose:** Ensure you never get unusable output without visibility into what went wrong.

## CopySnap: Fast Editing & Replies

CopySnap is a lightweight, fast tool for quick text improvements, replies, and question generation.

**Access:** Top navigation → "CopySnap" or `/copy-snap` route

**Key Differences from Copy Maker:**
- Simpler interface (fewer fields)
- Faster generation (optimized prompts)
- 2000 character input limit
- Focused on quick tasks
- Lower credit cost per operation

---

### CopySnap: Improve Mode

**Purpose:** Polish and enhance existing text quickly.

**When to use:**
- Tighten up a draft
- Make content clearer or more persuasive
- Adapt content for different platforms
- Quick rewrites without full Copy Maker setup

#### Inputs

**Text to Improve (required):**
- Paste the content you want to improve
- Character limit: 2000
- Live character counter shown

**Goal:**
- **Clearer:** Simplify language, improve clarity
- **Persuasive:** Boost convincing power
- **Shorter:** Reduce word count (30-50%)
- **Punchier:** Make more impactful and memorable

**Platform:**
- **General:** No platform-specific optimization
- **X (Twitter):** Consider 280-char limit, conversational tone
- **LinkedIn:** Professional, thought-leadership angle
- **Email:** Clear subject, compelling CTA

**Length:**
- **Short:** 30-50% reduction from original
- **Same:** ±10% of original length
- **Longer:** +20-50% expansion

**Special Instructions (optional):**
- Additional guidance for AI
- Examples: "Use active voice," "Avoid jargon," "Add humor"

**Human Tone Toggle:**
- **OFF:** Standard AI output
- **ON:** More conversational, natural phrasing
- Reduces AI detection patterns
- Avoids buzzwords and clichés

#### Output

**Structure:**
```
Best Version: [Primary improved text]

Alternatives: [2 additional variations]

Notes: [3 improvement tips explaining changes]
```

**What you get:**
- 1 primary improved version
- 2 alternative versions (different approaches)
- 3 improvement notes (what was changed and why)

#### Post-Generation Actions (Improve Mode)

**Copy to Clipboard:** Copy any version instantly
**Replace Input:** Click to use an alternative as new input
**Modify:** Add modification instructions and regenerate
**Save Output:** Save to Saved Outputs library
**Start Over:** Clear input and output

### CopySnap: Answer Mode

**Purpose:** Generate thoughtful, production-ready replies to messages, comments, or posts.

**When to use:**
- Respond to social media comments
- Reply to emails or messages
- Engage with LinkedIn posts
- Answer forum questions

#### Inputs

**Message to Reply To (required):**
- Paste the message/comment you're responding to
- Character limit: 2000

**Reply Style:**
- **Helpful:** Adds concrete ideas, no filler
- **Friendly:** Warm tone, max 1 emoji, no sarcasm
- **Confident:** Clear and assertive
- **Witty:** Clever humor (never sarcastic)
- **Direct:** Concise, no emojis, statements over questions

**Stance:**
- **Neutral:** Balanced, observational perspective
- **Agree:** Supports original message, adds original insight
- **Disagree:** Respectfully challenges (always starts with acknowledgment)

**Length:**
- **Short:** 1-2 sentences, max 1 question
- **Medium:** 2-4 sentences, max 1 question
- **Long:** 4-6 sentences, max 2 questions, includes concrete insight

**Special Instructions (optional):** Additional guidance

**Human Tone Toggle:** Same as Improve mode

#### Output

**Structure:**
```
Best Reply: [Primary response]

Alternatives: [2 additional variations]
```

**What you get:**
- 1 primary reply
- 2 alternative replies (different angles)

**Validation:** Answer mode has strict 10-point quality checks:
1. Reply is coherent standalone
2. Adds value beyond restating original
3. Respects selected stance
4. Matches selected style
5. Sentence count within bounds
6. Question count within bounds
7. Emoji count within limits
8. No inappropriate hashtags
9. No markdown formatting
10. Ready to post without editing

**Fallback System:** If output fails validation, AI automatically regenerates with fallback settings (Helpful + Neutral + Medium).

#### Post-Generation Actions (Answer Mode)

**Copy to Clipboard:** Copy any reply
**Replace Input:** Use alternative as new base
**Modify:** Refine reply with additional instructions
**Save Output:** Archive for future reference

## Credits & Usage

CopyZap uses a credits-based system. All AI operations consume credits.

---

### What Are Credits?

**Credits = Cost Unit**
- 1 credit = $0.01 USD
- Every AI request consumes credits based on:
  - Model used
  - Tokens processed (input + output)
  - Operation type

**How credits are calculated:**
1. AI returns token count (input, output, reasoning)
2. System looks up pricing for model (from database)
3. Calculates raw cost in USD: `cost = (input_tokens * input_rate) + (output_tokens * output_rate)`
4. Applies billing markup: `adjusted_cost = raw_cost * 1.30` (30% markup)
5. Converts to credits: `credits = ceil(adjusted_cost / 0.01)`
6. Minimum charge: 1 credit

**Example:**
```
Model: GPT-4o
Input tokens: 1500
Output tokens: 2000
Pricing: $0.015/1k input, $0.06/1k output

Raw cost: (1500 * 0.015 / 1000) + (2000 * 0.06 / 1000) = $0.1425
Adjusted: $0.1425 * 1.30 = $0.18525
Credits: ceil(0.18525 / 0.01) = 19 credits
```

---

- **Duration:** 30 days from registration

**Billing Period:**
- Based on `credits_period_start_day` (day of month you signed up)
- Resets monthly
- Example: Sign up Jan 15 → Period is 15th to 15th each month

**Running Low on Credits:**
- Yellow indicator at 50-90% used
- Red indicator at >90% used
- Warning toast at 95% used

### What Happens When You Run Out of Credits?

**During Free Trial:**
- Credits = 0 means unlimited
- No enforcement

**After Free Trial (Paid Plan Required):**
- Generation blocked
- "Out of credits" modal appears
- Options:
  - Purchase more credits (link to billing)
  - Wait until next billing period (if monthly plan)

**Grace Credits:**
- Admin can assign "grace credits" (temporary buffer)
- Allows negative balance temporarily
- Example: 100 credits allowed + 20 grace = can use up to 120 before blocking

### Output Validation & Repair Credits

**When AI generates invalid output:**
- System automatically attempts repair
- Repair tracked as separate operation: `generate_copy_repair`
- Repair charged same rate as original generation
- **Effectively doubles cost if repair is needed**

**Example:**
```
Original generation: 15 credits
Validation fails (missing content)
Repair attempt: 15 credits
Total: 30 credits
```

**No refunds for failed generation** — both original and repair consume credits.

**How to avoid repair costs:**
- Provide clear, complete inputs
- Avoid placeholder variables in required fields
- Use proven templates
- Test with cheap models first

## Troubleshooting & FAQs

### Generation Issues

**Q: Generation is stuck/frozen — what do I do?**
A: Wait 60 seconds. If still stuck, refresh the page. Session auto-saves, so you won't lose inputs. Retry generation.

**Q: I got "Out of credits" but I'm on free trial.**
A: Check your registration date. Free trial is 30 days. After that, credits = 0 and generation blocks. Contact admin to assign credits.

**Q: AI generated content with placeholders still in it (e.g., `[CLIENT_NAME]`).**
A: You likely had placeholders in your inputs. CopyZap warns you but allows proceeding. Fix inputs and regenerate, or use "Modify" action to replace placeholders.

**Q: Output is in the wrong language.**
A: Check Language setting in Copy Maker. If set to Auto-detect, AI detects from your inputs. If inputs are mixed languages, AI may guess wrong. Set language manually.

**Q: Output is too short/too long despite word count setting.**
A: Enable "Prioritize Word Count" toggle. This forces AI to revise until target is met (adds cost).

**Q: AI ignored my special instructions.**
A: Instructions may be too vague or contradictory. Be specific. Example: Don't say "make it good" — say "use active voice and avoid jargon."

**Q: Generation failed with no output.**
A: Validation failed and repair also failed. Check validation warning banner. Common causes:
- Inputs too vague
- Model overloaded (rare)
- Conflicting settings

Click "Show Raw Output" to see what AI returned. Adjust inputs and retry.

---

### Credit Issues

**Q: Why did one generation cost 30 credits when another cost 15?**
A: Cost varies by:
- Output length (more tokens = more cost)
- Model used (Claude more expensive than DeepSeek)
- Repair attempts (doubles cost if validation fails)
- Optional features (SEO, scoring, GEO add separate charges)

**Q: Can I get refund for bad output?**
A: No. Credits are consumed on API request, not on output quality. Even failed generations (validation errors, repair attempts) consume credits.

**Q: How do I know how many credits a generation will cost BEFORE running it?**
A: You don't — cost depends on output length (unknown until generated). Estimate based on target word count:
- 500 words with GPT-4o: ~15 credits
- 1000 words with GPT-4o: ~30 credits
- CopySnap operations: 2-8 credits

Check Credits Usage tab after a few generations to understand your typical costs.

**Q: I see "generate_copy_repair" in my usage log — what is that?**
A: Automatic repair attempt after validation failure. Charged same as original generation. Prevent this by using clear inputs and tested templates.

---

### Output Quality Issues

**Q: AI output sounds robotic or generic.**
A: Enable "Human Tone" toggle (CopySnap) or create a detailed Brand Voice (Copy Maker). Provide more context in inputs. Use higher-quality models (Claude, GPT-4o).

**Q: AI keeps generating marketing buzzwords I don't want.**
A: Create Brand Voice with Forbidden Terms list. Add unwanted words: "synergy," "leverage," "game-changing," etc.

**Q: How do I make output more creative/unique?**
A: Increase temperature (advanced setting, not exposed in UI by default). Or use "Modify" action with instruction "make more creative and less conventional."

**Q: Output doesn't match my brand voice even though I selected it.**
A: Review brand voice profile. May need more specific rules. Test by generating multiple times and refining voice profile based on patterns you see.

**Q: AI keeps using British English but I want American English.**
A: Add to Special Instructions: "Use American English spelling and terminology." Or add to Brand Voice as preferred vocabulary: "color" (not "colour"), "optimize" (not "optimise"), etc.

---

### Template & Session Issues

**Q: I loaded a template but nothing happened.**
A: Templates pre-fill fields but do NOT auto-generate. After loading, review fields and click "Generate Copy."

**Q: I can't find my saved session.**
A: Check Dashboard → Sessions tab. Use search or filter by customer. Sessions are auto-saved immediately after generation.

**Q: How do I delete multiple sessions at once?**
A: Currently not supported. Delete one at a time.

**Q: I edited a template but old sessions still use old configuration.**
A: Correct. Sessions snapshot configuration at generation time. Editing template doesn't retroactively change past sessions.

---

### Workflow Issues

**Q: Workflow failed midway — do I get credits back?**
A: No. Each completed step consumes credits even if later steps fail.

**Q: Can I pause a workflow and resume later?**
A: No. Workflows run to completion once started.

**Q: Workflow output doesn't match my expectations.**
A: Review each step's configuration. Test individual steps before chaining into workflow. Workflows are only as good as their step instructions.

**Q: I see a public workflow I want to customize — can I edit it?**
A: No (unless you're the creator). Run the workflow, then create your own based on the results. Or duplicate by recreating steps manually.

---

### Brand Voice Issues

**Q: AI-generated brand voice doesn't match my samples.**
A: Provide more samples (5-10 examples). Or manually edit generated profile to emphasize specific traits.

**Q: I have multiple brand voices — how do I choose the right one?**
A: Name them descriptively: "Company Blog Voice," "CEO LinkedIn Voice," "Sales Email Voice." Load appropriate voice based on content type.

**Q: Can I apply brand voice to CopySnap outputs?**
A: Not directly. CopySnap doesn't have brand voice selector. Instead, load CopySnap output into Copy Maker, select brand voice, and regenerate.

---

### Export & Integration Issues

**Q: How can I create a PDF of my content?**
A: Export as HTML, open it in your browser, and use File → Print → Save as PDF. This gives you full control over formatting.

**Q: Markdown export formatting is broken when I paste into [tool].**
A: Different tools handle Markdown differently. Try exporting as HTML if Markdown doesn't render correctly.

**Q: Can I export directly to WordPress/HubSpot/etc.?**
A: No. Export as HTML or Markdown and paste into your CMS manually. No direct integrations available.

**Q: FAQ Schema isn't being recognized by Google.**
A: Ensure you're using FAQ output structure. Validate JSON-LD with Google's Rich Results Test tool. Schema generation is only available when FAQ format is used.

---

### Account & Access Issues

**Q: I can't log in with Google OAuth.**
A: Google OAuth is restricted to admin only. Create account using email/password.

**Q: I didn't receive a welcome email.**
A: Email confirmation is disabled by default. Check spam folder or contact admin if you believe you should have received one.

**Q: Can I use CopyZap on mobile?**
A: Not optimized for mobile. Desktop browser required for full functionality.

**Q: Do I need to manually save sessions?**
A: No. All Copy Maker sessions auto-save after generation. CopySnap does NOT auto-save — use "Save Output" if you want to keep results.

## Glossary

**Alternative Copy / Alternative Version**
Additional variations of generated content. Each alternative explores different messaging angles or styles while addressing the same brief.

**Analyze & Compare**
Workflow step or standalone action that uses AI to evaluate multiple content versions, compare strengths/weaknesses, and recommend best option.

**Blended Copy**
Output created by merging the best elements of multiple generated versions using AI.

**Brand Voice**
Saved profile defining personality, tone, vocabulary, sentence structure, and punctuation rules for consistent content generation.

**Brief Description**
Short summary of what you're creating. Acts as session name and provides high-level context to AI.

**Billable Units**
Internal term for credits. 1 billable unit = 1 credit = $0.01 USD.

**CopySnap**
Fast, lightweight text improvement tool with three modes: Improve, Answer, Question. Lower cost than Copy Maker, 2000-char limit.

**Copy Maker**
Full-featured content generation engine with 80+ configuration options. Main tool for creating marketing copy from scratch or improving existing text.

**Credits**
Usage currency. All AI operations consume credits based on model, tokens, and operation type. 1 credit = $0.01 USD.

**CTA (Call to Action)**
Specific instruction telling readers what action to take (e.g., "Sign up now," "Download the guide").

**Customer**
Client or business entity (used by agencies to organize content per client).

**Dashboard**
Command center showing sessions, templates, saved outputs, credit usage, and workflows.

**DeepSeek**
Cost-effective AI model used as fallback when primary model fails. Also available as primary selection for budget-conscious users.

**Enforcement Mode**
System that blocks generation when credits are exhausted (applies after free trial).

**FAQ Schema (JSON-LD)**
Structured data format for FAQ pages that helps search engines display rich results.

**Floating Action Bar**
Toolbar that appears after generation with export, save, and modification options.

**Generate Copy**
Primary action button that initiates AI content generation.

**GEO Score (Generative Engine Optimization)**
AI-based evaluation of content's likelihood to appear in AI assistant responses (ChatGPT, Perplexity, etc.).

**Grace Credits**
Temporary credit buffer allowing negative balance before enforcement kicks in.

**Grok**
AI model by X.AI available for content generation and analysis.

**Humanized Version**
Output rewritten to sound more conversational and less AI-generated.

**Improve Mode**
Copy Maker mode for refining existing text (vs. creating from scratch).

**Model**
AI engine used for generation (GPT-4o, Claude, DeepSeek, Grok, Gemini). Different models have different costs and strengths.

**Modify Content**
Post-generation action or workflow step that applies AI transformation based on instruction (e.g., "make more casual").

**Output Structure**
Format of generated content: Paragraph, Bulleted List, Numbered List, FAQ, Structured Sections.

**Output Validation**
System that checks generated content for structural completeness and quality. Triggers automatic repair if validation fails.

**Prefill**
Saved snapshot of input fields (similar to template but includes all data, not just configuration).

**Prompt Evaluation**
AI's assessment of your input quality with suggestions for improvement.

**Public Template/Workflow**
Template or workflow visible to all users (created by admin).

**Repair**
Automatic retry attempt when output fails validation. Tracked as `generate_copy_repair` operation and charged separately.

**Restyle / Apply Persona**
Apply different brand voice to existing output without full regeneration.

**Saved Output**
Manually saved content stored in Dashboard → Saved Outputs tab.

**Score / Content Score**
AI-generated quality metrics: Clarity, Persuasiveness, Engagement, Readability (0-100 scale).

**Session**
Snapshot of one Copy Maker generation including all inputs, outputs, and settings. Auto-saved after generation.

**SEO Metadata**
Search-optimized meta titles and descriptions generated for blog posts and web pages.

**Special Instructions**
Free-text field for additional AI guidance that doesn't fit other input fields.

**Start Hub**
Modal overlay offering three entry paths: Quick Wizard, Templates, or Scratch.

**Structured Sections**
Output format with custom-labeled sections (e.g., Hero, Features, Benefits) each with target word counts.

**Target Audience**
Description of who will read the content. Influences language, tone, and messaging.

**Template**
Saved Copy Maker configuration (all input fields) for reuse. Does NOT include generated output.

**Tokens**
Units of text processed by AI. Cost is based on input tokens + output tokens + reasoning tokens.

**Tone & Style**
Voice characteristics (Professional, Casual, Authoritative, etc.) applied to generated content.

**USP (Unique Selling Proposition)**
What makes your product/service distinct from competitors.

**URL Analysis**
Feature that scrapes competitor or your own website to extract context for generation.

**Workflow**
Sequence of automated steps (template application, generation, voice transformation, analysis) for complex content processes.

**Word Count Tolerance**
Percentage variance allowed from target word count (default 10%). Example: 500-word target with 10% tolerance = accept 450-550 words.
