# CopyZap - Saved Templates & Prefills System

## Overview

The Saved Templates system allows users to save successful form configurations as reusable templates, enabling instant setup for similar projects without re-entering fields.

---

## What Are Templates?

**Templates = Complete Form State Snapshots**

When you save a template, you capture:
- All 40+ input field values
- Mode selection (Create/Improve)
- AI model choice
- Tone, word count, structure settings
- SEO/GEO configurations
- Special instructions
- Output structure definitions
- All advanced settings

**NOT Saved:**
- Generated output content
- Customer selection
- Brief/Project description (organizational fields)

---

## Creating Templates

### From Main Form

**Method 1: Save After Configuration**
1. Fill out form completely
2. Click "Save as Template" button
3. Enter template name
4. Optionally add description
5. Save

**Method 2: Save After Generation**
1. Generate content
2. Review results
3. If configuration worked well, save as template
4. Name it descriptively

### From Wizard

**After Wizard Completion:**
1. Complete wizard flow
2. Apply configuration to form
3. Review and adjust if needed
4. Save as template from main form

## Prefill Management

### What Are Prefills?

**Prefills = Field-Specific Saved Values**

Unlike templates (complete configurations), prefills save individual field values for reuse across different contexts.

**Common Prefill Uses:**
- Brand values (reuse across all projects)
- Company descriptions (standard about text)
- Target audiences (repeat customer profiles)
- Special instructions (brand style guides)

### Creating Prefills

**From Any Field:**
1. Fill field with value you'll reuse
2. Click "Save as Prefill" icon
3. Name the prefill
4. Save

**Example Prefills:**

**Brand Values Prefill:** "Our Core Values"
```
Transparency, Innovation, Customer Success, Sustainability, Simplicity
```

**Special Instructions Prefill:** "Vienna Style Guide"
```
Use Viennese dialect naturally. Reference local culture. Maximum 3 sentences
per paragraph. Bold product names. Use British English spelling.
```

**Business Description Prefill:** "Standard Company Intro"
```
We provide cloud-based project management software designed specifically for
remote teams of 5-50 people. Our platform solves coordination chaos...
```

### Using Prefills

**Dropdown Selection:**
1. Click prefill dropdown on field
2. Select saved prefill
3. Field populates instantly
4. Edit if needed for this specific use

**Benefits:**
- Consistency across projects
- Speed (no retyping)
- Accuracy (no transcription errors)
- Standardization across team

---

## Template vs Prefill: When to Use Each

### Use Templates When:
- ✅ Repeating entire content type
- ✅ Client-specific full configurations
- ✅ Standard project types
- ✅ Proven winning setups

### Use Prefills When:
- ✅ Reusing specific field values
- ✅ Maintaining brand consistency
- ✅ Standard descriptions/values
- ✅ Team collaboration (shared prefills)

### Use Both Together:
```
Load "Landing Page Template" →
Select "Vienna Style Guide" prefill for Special Instructions →
Select "Our Core Values" prefill for Brand Values →
Adjust product-specific details →
Generate
```
