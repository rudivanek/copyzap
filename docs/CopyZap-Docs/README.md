# CopyZap Documentation

This folder contains the source markdown documentation for CopyZap.

## Structure

- **CopyZap-Features.md** - Master comprehensive features documentation (ALWAYS update when adding/modifying features)
- **brand-voice-user-guide.md** - Brand Voice system user guide
- **CONSOLIDATED_ADDITIONS.md** - Consolidated feature additions
- **backups/** - Backup versions of documentation files

## Important

When adding or modifying ANY feature in the codebase, you MUST update `CopyZap-Features.md` with:
1. Accurate feature description
2. Proper section placement
3. Consistent formatting
4. Updated timestamp in metadata

## HTML Documentation

The user-facing HTML help documentation is located in `/docs/*.html` (parent directory).
